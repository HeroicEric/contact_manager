# Contact Manager

## Contact

- Age
- Blood type
- Email
- First name
- Height
- Home address
- Last name
- Phone number

## User Stories

As a user
I want to see all of my contacts
So that I can contact them

Acceptance Criteria:

- I should see each contact's name
- Contact's name should be displayed "last, first"
- Contacts should be ordered alphabetically by last name
